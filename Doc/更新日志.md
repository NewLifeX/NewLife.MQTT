# 更新日志（CHANGELOG）

## 发布基线
- 上一次发布版本：`v2.3.2026.0201`（`2026-02-01`）
- 本次统计范围：`2026-02-02` 之后至 `2026-03-05`
- 提交总数：`22`

## 核心更新

### 1) 协议与功能增强
- 完善 MQTT 5.0 协议能力，增强服务端与客户端能力。
- 增加客户端 MQTT 版本显式指定能力，提升与不同 Broker 的兼容性。
- 新增 EMQX 共享订阅支持。

### 2) 集群与稳定性改进
- 修复集群相关缺陷并补充 MQTT 5.0 特性单元测试。
- 修正 `MqttController.OnConnect` 返回类型与基类签名一致性。
- 根据代码评审反馈补充异常日志，并优化管理服务测试端口策略（`Port=0`）。
- 完善集群自动发现与管理 API。

### 3) 兼容性与工程能力提升
- 提升 .NET 多目标框架兼容性，优化异步与安全相关实现。
- 为 .NET 7/8 显式启用 QUIC 预览能力（`EnablePreviewFeatures`），保障相关功能可用。
- 升级核心依赖：
  - `NewLife.Core` 至 `11.13.2026.301`
  - `NewLife.Remoting` 至 `3.7.2026.302`
  - `NewLife.Stardust` 至 `3.7.2026.303`
- 多次执行 NuGet 包例行升级，持续保持依赖新鲜度。

### 4) 文档与元数据优化
- 更新分析报告与架构相关文档，统一传输端口说明，优化协议描述与规范。
- 优化项目元数据与包描述信息，提升分发可读性。
